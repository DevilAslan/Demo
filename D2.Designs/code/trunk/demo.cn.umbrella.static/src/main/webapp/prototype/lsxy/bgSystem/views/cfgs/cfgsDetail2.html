主要违法事实：叶增锚，浙江省文成县人，现在温州市信河街3号经营经营一家食品行，持有烟草专卖零售许可证，证号：33030120150514354A，国家局编码：330301143296。前段时间，一自称姓“余”的外地男子来被处罚人叶增锚店里要求高价收购软中华，被处罚人叶增锚觉得有利可图，便和他相互留了联系电话。临近年底，被处罚人从烟草公司购得五十余条软中华，就电话联系了余先生，双方在电话里谈好价格并把卷烟送往余姚交易。被处罚人叶增锚感觉仅凭自己店里的中华烟送过去利润不大，于是又到大南门的“某某烟酒行”购进一些软中华。被处罚人叶增锚从烟草公司及某某烟酒行处均以583元/条的价格合计购得中华（软）84条。2016年2月4日上午十点多，被处罚人叶增锚在未办理烟草专卖品准运证的情况下，携带装有该批卷烟的两个旅行袋乘坐公交车前往温州动车南站，在温州动车南站大厅的自动售票机上购买了前往余姚的动车票，在动车南站进站安检口准备进站时被我局执法人员查获。按购买价格计算，叶增锚运输卷烟价值达48972元。经浙江省烟草质量监督检测站鉴别检验，上述被先行登记保存的卷烟均为真品卷烟。

处罚依据：根据《中华人民共和国烟草专卖法》第二十九条第一款、《中华人民共和国烟草专卖法实施条例》第五十四条第（一）项之规定。

处罚结果：一、处以违法运输的卷烟价值48972元42%的罚款，计20568.24元上缴国库；二、将违法运输的卷烟予以收购；三、依据《中华人民共和国烟草专卖法实施条例》第六十八条之规定，卷烟收购款34280.4元予以返还。

履行方式与期限：被处罚人应当在收到本处罚决定书之日起十五日内到中国建设银行温州支行（或分理处）缴纳罚款（收款人名称：行政罚没款暂挂户，收款人账号：10133062353524103500907050）。按照《中华人民共和国行政处罚法》第五十一条第（一）项、第（三）项规定：1、到期不缴纳罚款的，每日按罚款数额的百分之三加处罚款；2、逾期不履行本处罚决定的，申请人民法院强制执行。如不服本处罚决定，可在收到本处罚决定书之日起60日内向浙江省烟草专卖局申请复议，或在收到本处罚决定书之日起15日内向温州市鹿城区人民法院起诉。该行政处罚决定书自送达之日起生效。
