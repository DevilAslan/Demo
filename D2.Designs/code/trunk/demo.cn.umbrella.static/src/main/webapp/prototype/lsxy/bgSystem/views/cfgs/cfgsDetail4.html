主要违法事实：阮冬祥，浙江省温岭市人，在温州市瓯海区梧田月乐西街149号经营一家副食品店。持有烟草专卖零售许可证,证号：33030120141029541A，国家局编码: 330301142487，持有个体工商户营业执照,注册号：330304604223966。被处罚人阮冬祥于2016年2月3日上午许一名上门兜售卷烟的男子处以100元/条的价格购入中华（软）1条，以80元/条的价格购入中华（硬）2条，总计2个品种3条。货款当场以现金结清，置于店中销售。违法销售总额260元。经浙江省烟草质量监督检测站鉴别检验，上述被先行登记保存的卷烟为假冒注册商标且伪劣卷烟（非法生产）。

处罚依据：根据《中华人民共和国烟草专卖法实施条例》第六十一条

处罚结果：一、责令停止销售非法生产的卷烟；二、处以违法销售总额260元45%的罚款，计117元上缴国库；三、将涉案的假冒注册商标且伪劣卷烟(非法生产)3条予以公开销毁；四、鉴于被处罚人因违法生产经营烟草专卖品一年内被烟草专卖行政主管部门或其他执法机关处罚两次以上，根据《中华人民共和国烟草专卖法实施条例》第十六条，《烟草专卖许可证管理办法》第四十五条第（三)项之规定，取消其从事烟草专卖业务的资格。

履行方式与期限：被处罚人应当在收到本处罚决定书之日起十五日内到中国建设银行温州支行（或分理处）缴纳罚款（收款人名称：行政罚没款暂挂户，收款人账号：10133062353524103500907050）。按照《中华人民共和国行政处罚法》第五十一条第（一）项、第（三）项规定：1、到期不缴纳罚款的，每日按罚款数额的百分之三加处罚款；2、逾期不履行本处罚决定的，申请人民法院强制执行。如不服本处罚决定，可在收到本处罚决定书之日起60日内向浙江省烟草专卖局申请复议，或在收到本处罚决定书之日起15日内向温州市鹿城区人民法院起诉。该行政处罚决定书自送达之日起生效。
