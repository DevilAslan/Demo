行政处罚决定书文号：温龙市监处字〔2015〕260号；
      案件名称：任美英生产、销售不符合保障人体健康和人身、财产安全的国家标准、行业标准的产品案；
      被处罚对象名称：任美英；
      被处罚单位法定代表人姓名：未录入；
      主要违法事实：本局在开展食品安全监测专项行动中，对位于温州市龙湾区永中街道永中农贸市场的温州市龙湾永中任美英熟食摊销售的烧鹅（熟肉制品）、鸭掌（熟肉制品）进行抽样检验，上述两种食品各抽取0.75kg作抽样样品，抽取0.25kg作备份样品。在当事人收到上述二种食品的检验报告之前，上述二种同批次食品已经销售完毕，其中烧鹅（熟肉制品）购进5kg，进货单价46元/kg，销售单价50元/kg（其抽样样品与备份样品以进货单价销售），销售5kg，经营额为246元，获利16元；鸭掌（熟肉制品）购进3kg，进货单价46元/kg,销售单价50元/kg（其抽样样品与备份样品以进货单价销售），销售3kg，经营额为146元，获利8元。上述二种食品的经营额合计为392元，获利合计为24元。上述食品经温州市质量技术监督检测院抽样检验，结论显示：烧鹅（熟肉制品）的“菌落总数”、“大肠菌群”项目、鸭掌（熟肉制品）的“大肠菌群”项目实测结果不符合标准要求。 ；
      行政处罚种类、依据：生产、销售不符合保障人体健康和人身、财产安全的国家标准、行业标准的产品；
      行政处罚履行方式：处罚决定；
      行政处罚履行期限：2016-01-21 00:00:00.0；
      行政处罚机关名称：温州市龙湾区区市场监管局；
