任何单位和个人随意倾倒、抛撒或者堆放建筑垃圾的处罚

2015年9月5日至9月6日22时10分，当事人温州市宏盛建筑基础工程有限公司，将仙岩街道沈岙三产安置房工地地下室开挖的建筑垃圾（粘土）外运，并在瓯海区仙岩街道仙岩第二中学东南侧空地进行倾倒消纳，倾倒建筑垃圾的方量约为1200立方米，现场出示建筑垃圾处置准运证核准的路线为：工地-104国道-罗山大道-海工大道-滨海大道-金海大道-堤坝-瓯飞起步区，消纳地点为瓯飞起步区。实际运输路线为：工地—104国道—下沈线-消纳后场，处置消纳的地点为瓯海区仙岩街道仙岩第二中学东南侧空地，在瓯海区仙岩街道仙岩第二中学东南侧空地有随意倾倒、抛撒建筑垃圾行为。该行为已违反了《城市建筑垃圾管理规定》第十五条之规定。违法性质为：随意倾倒、抛撒建筑垃圾。证据有：现场检查（勘验）笔录1份;调查（询问）笔录1份;现场违法照片3份6张;温州市建筑垃圾处置准运证复印件1份（NO:1217260)；等为凭 。 
    整改届满后，执法人员于2015年9月15日对现场进行复查，当事人未对倾倒的建筑垃圾进行清理。当事人未改正违法行为。当事人此次随意倾倒、抛撒建筑垃圾系今年第二次实施该类别违法行为。
    鉴于当事人整改届满后未改正违法行为且一年内多次实施同一类别违法行为，屡教不改，按照《温州市城市管理行政处罚自由裁量权行使规定》第十条第一款第（一）项、第（二）项之规定，予以从重处罚。依据《城市建筑垃圾管理规定》第二十六条之规定，责令改正违法行为，警告并对处以罚款人民币肆万玖仟捌佰元整的行政处罚。

限你单位自收到本处罚决定书之日起15日内，将罚款缴至瓯海农村合作银行营业网点。逾期不缴纳罚款，依据《中华人民共和国行政处罚法》第五十一条第一项规定每日按罚款数额3%加处罚款（加处罚款最高数额为罚款本金数额）。
你单位如不服本处罚决定，可在收到本处罚决定书之日起60日内向瓯海区人民政府或温州市城市管理与行政执法局申请行政复议，也可以在六个月内向瓯海区人民法院起诉。行政复议和行政诉讼期间不停止本处罚决定的执行。
逾期不申请行政复议，也不提起行政诉讼，又不履行行政处罚决定的，本机关将依法申请人民法院强制执行。
